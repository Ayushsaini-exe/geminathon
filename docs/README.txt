Place your agricultural documents here.

Supported formats:
  - .txt  (plain text)
  - .md   (markdown)
  - .pdf  (requires: pip install pdfplumber OR pip install pypdf)

Subdirectories are also scanned.

To ingest into the RAG knowledge base, run:
  python -m scripts.ingest_files

To clear existing data and re-ingest:
  python -m scripts.ingest_files --clear

Examples of good documents to add:
  - Government crop advisory PDFs
  - ICAR/IARI research papers
  - State-specific farming guidelines
  - Pesticide usage manuals
  - Soil management guides
  - Weather-based crop calendars
